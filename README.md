## Описание программы 

Weather Diary - это GUI-приложение для ведения дневника погоды. Приложение позволяет добавлять метеорологические записи, фильтровать их по дате и температуре, сохранять и загружать данные в формате JSON, а также визуально отслеживать погодные наблюдения.
Возможности:

Добавление записей с указанием даты, температуры, описания погоды и наличия осадков


Фильтрация записей по дате (конкретный день)


Фильтрация по температуре (выше заданного значения, например, выше +10°C)


Сохранение данных в JSON-файл


Загрузка данных из JSON-файла


Просмотр всех записей в табличном виде


Валидация вводимых данных (проверка формата даты, числовое значение температуры, непустое описание)

Инструкция по запуску
Требования:

Python 3.6 или выше


Установленная библиотека Tkinter (обычно идёт в комплекте с Python)

Запуск:

Откройте терминал или командную строку


Перейдите в директорию проекта:

bash

cd weather_diary


Запустите приложение:

bash

python weather_diary.py

Примеры использования
Добавление записи о погоде:

Заполните все поля формы:


Дата: "2026-04-29" (формат: ГГГГ-ММ-ДД)


Температура: "15.5" (в градусах Цельсия)


Описание погоды: "Солнечно, легкий ветер"


Осадки: Отметьте галочку, если были осадки


Нажмите кнопку "Добавить запись"


Запись появится в таблице внизу окна

Фильтрация:

По дате: Введите дату в поле фильтра (например, "2026-04-29") и нажмите "Применить фильтр"


По температуре: Введите значение (например, "10") - покажет записи с температурой выше +10°C


Можно комбинировать оба фильтра одновременно


Для сброса нажмите "Сбросить фильтр"

Сохранение и загрузка:

Нажмите "Сохранить в JSON" - данные сохранятся в файл weather_data.json


Нажмите "Загрузить из JSON" - данные загрузятся из файла weather_data.json

Тестовые данные (примеры для проверки):
Попробуйте добавить следующие записи для тестирования:

Дата: 2026-04-25, Температура: 8.5, Описание: Пасмурно, холодный ветер, Осадки: Да


Дата: 2026-04-26, Температура: 12.0, Описание: Облачно, небольшой дождь, Осадки: Да


Дата: 2026-04-27, Температура: 18.5, Описание: Ясно, тепло, Осадки: Нет


Дата: 2026-04-28, Температура: 22.0, Описание: Солнечно, жарко, Осадки: Нет


Дата: 2026-04-29, Температура: 15.5, Описание: Легкая облачность, Осадки: Нет

После добавления попробуйте:

Отфильтровать записи за конкретный день (например, 2026-04-28)


Найти все дни с температурой выше +15°C


Сохранить данные в JSON, закрыть приложение, затем загрузить обратно

Валидация данных
Приложение проверяет:

Все поля должны быть заполнены


Дата должна быть в формате ГГГГ-ММ-ДД (например, 2026-04-29)


Температура должна быть числом (целым или дробным)


Описание не может быть пустым

Используемые технологии

Python 3 - язык программирования
Tkinter - стандартная GUI-библиотека Python
JSON - формат хранения данных
Git - система контроля версий
